HYPER TEXT MARKUP LANGUAGE

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar e estruturar páginas na web. Ele serve como a base de qualquer página da internet, sendo responsável por definir o conteúdo e a estrutura da página, como textos, imagens, links, tabelas, formulários e outros elementos.

Estrutura do HTML

Um documento HTML é composto por tags (marcadores) que indicam o tipo de conteúdo ou o comportamento que cada parte do documento deve ter. As tags são envolvidas por colchetes angulares < > e geralmente vêm em pares, com uma tag de abertura e uma de fechamento.

Exemplo de uma estrutura básica de HTML

            
        <!DOCTYPE html>
        <html>
            <head>
                <title>Minha Página</title>
            </head>
            <body>
                <h1>Bem-vindo ao meu site!</h1>
                <p>Este é um exemplo de uma página em HTML.</p>
            </body>
        </html>

Funções principais do HTML

  1. Estruturação do conteúdo: Define a organização de títulos, parágrafos, listas, links, imagens e outros elementos que compõem uma página web.
  2. Exibição de conteúdo: Fornece a base para a exibição de texto e mídia em navegadores da web.
  3. Interatividade: Combinado com linguagens como CSS (para estilos visuais) e JavaScript (para interatividade), o HTML permite criar páginas dinâmicas e interativas.
  4. Navegação: Usa links (<a>) para criar conexões entre diferentes páginas e sites.

    HTML é essencial para o funcionamento da web, servindo como a fundação sobre a qual todas as outras tecnologias de front-end são construídas.

CASCADING STYLE SHEETS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para controlar a aparência visual de páginas web escritas em HTML. Ele define como os elementos de uma página devem ser exibidos, como cores, fontes, espaçamento, layout e outras propriedades de design.

Função do CSS

Enquanto o HTML é responsável pela estrutura e conteúdo da página, o CSS é usado para a aparência e o estilo, tornando a página visualmente atraente e organizada.

Exemplo básico de CSS

Aqui está um exemplo de como o CSS pode ser usado para estilizar uma página HTML:

            
        <!DOCTYPE html>
        <html>
        <head>
        <style>
            body {
            background-color: lightblue;
            }
            h1 {
            color: navy;
            text-align: center;
            }
            p {
            font-family: Arial, sans-serif;
            font-size: 16px;
            }
        </style>
        </head>
        <body>
        <h1>Bem-vindo ao meu site!</h1>
        <p>Este é um exemplo de como o CSS
         pode alterar a aparência 
        de uma página.</p>
        </body>
        </html>

    Nesse exemplo, o CSS define que

  • O fundo da página (body) será azul claro.
  • O título (h1) terá cor azul marinho e será alinhado ao centro.
  • O parágrafo (p) usará a fonte Arial com um tamanho de 16 pixels.

    Principais funcionalidades do CSS

  1. Design visual: O CSS permite definir cores, fontes, espaçamento entre elementos, tamanhos e bordas, controlando a estética da página.
  2. Layout: Facilita a criação de layouts responsivos (adaptáveis a diferentes tamanhos de tela), como sistemas de colunas, alinhamento de elementos e posicionamento de blocos.
  3. Responsividade: CSS pode ser usado em conjunto com media queries para adaptar o design de acordo com o dispositivo (celular, tablet, desktop), melhorando a experiência do usuário.
  4. Reutilização de estilos: Com o CSS, os estilos podem ser definidos uma única vez e aplicados a vários elementos ou páginas, economizando tempo e garantindo consistência no design.
  5. Animações e transições: Permite criar efeitos visuais dinâmicos, como animações, transições suaves de cores ou movimentos de elementos na página.

    Como o CSS é usado

  • CSS Inline: Estilos são aplicados diretamente em elementos HTML, como no atributo style.
  • CSS Interno: Estilos são escritos dentro de uma tag <style> no próprio documento .
  • CSS Externo: Um arquivo CSS separado (.css) é vinculado ao documento HTML, facilitando a manutenção e a reutilização dos estilos em várias páginas.

    O CSS é fundamental para a criação de sites bonitos, acessíveis e agradáveis ao usuário.

FUNCIONALIDADE DAS TRES FUNÇÕES NO FRONT-END

JAVASCRIPT

JavaScript é uma linguagem de programação amplamente utilizada para adicionar interatividade e funcionalidades dinâmicas às páginas web. Diferente do HTML, que estrutura o conteúdo, e do CSS, que define o estilo, o JavaScript é responsável por tornar a página interativa e dinâmica, permitindo responder a ações do usuário e manipular dados em tempo real.

    Funções do JavaScript

  • Interatividade: Cria eventos, como cliques e digitação, permitindo que a página responda às ações do usuário, como exibir um pop-up ou validar um formulário.
  • Manipulação do DOM: Modifica a estrutura da página no navegador sem recarregá-la, permitindo adicionar, remover ou alterar elementos.
  • Controle de dados: Processa e manipula dados, como coleta de informações de formulários, armazenamento local e comunicação assíncrona com servidores (AJAX).
  • Animações e efeitos visuais: Cria animações complexas e interativas, controlando detalhes visuais além do que o CSS oferece.
  • Integração com APIs: Permite que páginas web se conectem a APIs externas, trazendo funcionalidades adicionais, como mapas ou dados em tempo real.
  • Desenvolvimento de jogos e aplicações: Cria jogos e Single Page Applications (SPA), que oferecem uma navegação fluida sem recarregar a página.

Exemplo básico de JavaScript

                
        <!DOCTYPE html>
        <html>
        <head>
        <title>Exemplo de JavaScript</title>
        </head>
        <body>
        <h1>Bem-vindo ao meu site!</h1>
        <button onclick="mostrarMensagem()">Clique aqui</button>

        <p id="mensagem"></p>

        <script>
            function mostrarMensagem() {
            document.getElementById("mensagem").innerHTML = "Olá! 
            Você clicou no botão.";
            }
        </script>
        </body>
        </html>

    Nesse exemplo

  • O botão, quando clicado, aciona a função mostrarMensagem().
  • A função altera o conteúdo do parágrafo com o id mensagem, exibindo uma mensagem "Olá! Você clicou no botão."

Resumo

JavaScript é uma linguagem poderosa e versátil, essencial para o desenvolvimento web moderno. Ele permite criar páginas dinâmicas e interativas, processar dados e se comunicar com servidores, proporcionando uma experiência rica e fluida ao usuário.
